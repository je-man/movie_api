Using the API
Show movies

Will show list of movies

GET http://localhost:8080/movies

Return data about a director by name

 GET http://localhost:8080/movies/director/Peter%20Docter
Will return data about genre by name GET http://localhost:8080/movies/genres/Thriller
Will delete a movie by title DELETE http://localhost:8080/movies/Up
Will show list of users GET http://localhost:8080/users
Will add new users POST http://localhost:8080/users
Will updates user information PUT http://localhost:8080/users/:Username
Will show list of users by username GET http://localhost:8080/users/:Username
allows user to add movie to favorites POST http://localhost:8080/users/:Username/movies/:MovieID
allows user to remove movie from favorites DELETE http://localhost:8080/users/:Username/Movies/:MovieID
Allows users to deregister by email DELETE http://localhost:8080/users/:Email
Allows users to delete by username DELETE http://localhost:8080/users/:Username
